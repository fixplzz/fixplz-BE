## 📃 개요
**도로 위 성동구**는 서울시 성동구의 공공시설(스마트 쉼터, 스마트 흡연부스 등)에 대한 민원 신고 서비스입니다. <br>
사용자는 QR코드를 통해 간편하게 서비스를 이용할 수 있습니다.
<br>
